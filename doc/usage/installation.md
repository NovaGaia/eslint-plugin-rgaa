# Installation

## Installation via npm

```bash
npm install --save-dev eslint-plugin-rgaa-html
```

## Installation via pnpm

```bash
pnpm add -D eslint-plugin-rgaa-html
```

## Installation via yarn

```bash
yarn add -D eslint-plugin-rgaa-html
```
